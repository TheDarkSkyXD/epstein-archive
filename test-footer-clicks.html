Footer Click Test - Epstein Archive

This page demonstrates the fix for the footer click blocking issue.

Problem: The absolute inset-0 div was blocking clicks on footer links because it covers the entire viewport.
Solution: Added pointer-events: none to make the overlay ignore mouse events, allowing clicks to pass through to elements underneath.

Test 1: WITHOUT pointer-events: none (Red Overlay)

This overlay will BLOCK clicks on footer links
Support on coff.ee The End Times Substack

Test 2: WITH pointer-events: none (Green Overlay)

This overlay will ALLOW clicks on footer links
Support on coff.ee The End Times Substack
How to test:
1. Try clicking the red footer links - they should NOT work (blocked by overlay)
2. Try clicking the green footer links - they SHOULD work (overlay ignores clicks)
3. The fix is applied by adding pointer-events: none to the overlay elements.
